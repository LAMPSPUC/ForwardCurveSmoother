After running the 'run.jl' file, this folder will contain three csv files:
- recontructed_prices.csv - CSV file with the prices estimated for each one of the contracts defined in contracts.csv;
- elementary_prices.csv - CSV file with the estimated forward curve for each trading date. This curve is composed by the elementary prices, which are contracts with the delivery period of a day. 
 - elementary_errors.csv - CSV file with the estimated smoothed errors.
